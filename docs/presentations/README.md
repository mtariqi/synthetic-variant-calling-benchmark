## Project Presentation Slides

This folder contains the presentation slides associated with the
**Synthetic Variant Calling Benchmark** project.

The slides summarize:
- Scientific motivation and background
- Synthetic somatic mutation framework
- Nextflow + Singularity HPC pipeline
- DeepVariant vs GATK comparison
- Key findings on coverage depth and variant detection
- Reproducibility considerations

These slides are derived directly from the analyses, scripts, and results
contained in this repository and are provided for transparency,
reproducibility, and educational purposes.

Citation:
Islam, M. T., Alimoradian, A., & Al-Amoudi, R. (2025).
Synthetic Variant Calling Benchmark. Zenodo. https://doi.org/10.5281/zenodo.17915965
